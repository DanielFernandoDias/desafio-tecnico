Sobre o Sistema

Bem-vindo ao nosso sistema de gerenciamento de parceiros e empresas externas. Desenvolvido com o intuito de simplificar e otimizar a gestão dessas entidades, nossa aplicação oferece uma variedade de funcionalidades e recursos para atender às necessidades de negócios modernos. Abaixo estão os principais aspectos e características do nosso sistema:

Objetivo

Nosso sistema foi criado com o objetivo de fornecer uma plataforma robusta e intuitiva para cadastrar, listar e gerenciar parceiros e empresas externas. Com uma interface amigável e recursos avançados, buscamos facilitar o processo de administração e colaboração entre essas entidades.

Tecnologia Utilizada

O sistema foi desenvolvido utilizando tecnologias modernas e frameworks populares no desenvolvimento web, incluindo Angular para o frontend e Node.js para o backend. Utilizamos também o framework Material Design para o layout e componentes visuais, garantindo uma experiência consistente para os usuários.

Funcionalidades Principais

  • Cadastro e Listagem de Parceiros e Empresas Externas: Nossa aplicação permite o cadastro detalhado de parceiros e empresas externas, incluindo informações como nome, descrição, repositório GIT, URL de documentação, entre outros.
  • Gerenciamento de Registros: Além do cadastro, oferecemos funcionalidades completas de gerenciamento, incluindo a listagem de todos os registros, paginação para facilitar a navegação e ações para editar e excluir registros individualmente.
  • Autenticação e Manutenção de Sessão: Implementamos um sistema de autenticação básico que permite aos usuários acessar o sistema através de um formulário de login. Além disso, oferecemos a opção de "manter conectado", que armazena informações do usuário localmente para facilitar o acesso futuro.
  • Compartilhamento de Dados: Para melhorar a experiência do usuário, implementamos um mecanismo de compartilhamento de links que preserva o estado da página, incluindo a paginação, ao compartilhar o link com outros usuários.
  • Integração com API:Nosso sistema se integra perfeitamente com uma API para persistência dos dados, garantindo segurança e confiabilidade na manipulação e armazenamento das informações.
